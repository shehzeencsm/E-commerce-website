#  SAPPHIRE BANGLES & JEWELS
##  Description
This is a PHP & MySQL based e-commerce website for selling jewelry and bangles online.
Users can browse products, add items to cart, and place orders through a checkout system.
The website is responsive and designed using Tailwind CSS.
##  Features
- Product listing and details page
- Add to cart functionality
- Checkout system
- Order form submission
- Responsive UI design
##  Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- Tailwind CSS
- XAMPP
